# Number Baseball (숫자 야구)

C 언어로 구현한 텍스트 기반 숫자 야구 게임입니다
컴퓨터가 생성한 서로 다른 숫자 3개를 추측하여 맞히는 콘솔 게임으로 난이도 선택과 스트라이크/볼 판정, 시도 횟수 기록 기능을 제공합니다.

## 게임 규칙

- 서로 다른 숫자 3자리를 맞히는 게임입니다
- 추측한 숫자를 입력하면 결과를 알려줍니다

| 판정 | 의미 |
|------|------|
| 스트라이크 (S) | 숫자와 자리(위치)가 모두 맞음 |
| 볼 (B) | 숫자는 있지만 자리가 틀림 |

예시 (정답이 1 2 3일 때)

```
1 4 5  →  1S 0B
2 3 1  →  0S 3B
1 2 3  →  정답
```

## 난이도

| 난이도 | 숫자 범위 |
|--------|-----------|
| Easy   | 1 ~ 9  |
| Normal | 1 ~ 20 |
| Hard   | 1 ~ 50 |

Hard 모드를 클리어하면 숨겨진 이스터에그가 출력됩니다

## 기능

- 메인 메뉴 (게임 설명 / 게임 시작 / 나가기)
- 게임 룰 설명 출력
- 난이도 선택 (Easy / Normal / Hard)
- 입력값 유효성 검사 (범위 / 중복 / 문자 입력 방지)
- 스트라이크, 볼 판정
- 시도 횟수 기록
- Hard 모드 클리어 시 이스터에그 출력

## 빌드 및 실행

Windows 환경에서 windows.h(Sleep 함수)를 사용합니다.

```bash
gcc project.c -o project.exe
project.exe
```

실행 파일과 같은 디렉터리에 egg.txt가 있어야 이스터에그가 정상 출력됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| project.c | 메인 소스 코드 |
| project.h | 함수 선언 헤더 |
| egg.txt   | 이스터에그 텍스트 |
